The RangeShifter platform

An eco-evolutionary modelling framework

The RangeShifter GUI

The RangeShifter platform was first published in Bocedi et al. (2014). The software is coded in C++ and packaged as a freely available Windows GUI. The graphical user interface is particularly useful for initial model exploration and for demonstration and teaching purposes. For more complex or computationally demanding simulations, it can also be run in Batch mode.

RangeShifter v1.0 is accompanied by extensive documentation. For getting acquainted with the software, we recommend to first read the manual to understand the conceptual underpinnings of RangeShifter, and then run the tutorials offered in the manual.

For help on the RangeShifter GUI please contact Greta or Steve.
