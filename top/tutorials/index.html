Tutorials

Both the RangeShifter GUI and the RangeshiftR package come with extensive documentation and tutorials. We also offer regular workshop on using the software, which we will also advertise on this website.

The tutorials lead you step by step through the example applications described in Bocedi et al. (2014):

  • Tutorial 1: range expansions and long-distance dispersal
  • Tutorial 2: patch connectivity using different dispersal and settlement rules
  • Tutorial 3: evolution of dispersal strategies during range shifting

We recommend to carefully read the manual to understand the conceptual underpinnings of RangeShifter. Data for running the tutorials can be downloaded here.

Tutorials for RangeShifter GUI

Please download the RangeShifter GUI and the manual. Tutorials for runnung the RangeShifter GUI are contained in chapter 4 of the manual. Additionally, the manual describes how tutorial 2 can be run in batch mode. This requires additional input parameter files that are documented here.

Tutorials for RangeshiftR package

We use the same three tutorials to introduce the different functionalities of the RangeshiftR package. All tutorials as well as guidance for installing the package can be found here.

Please note that the RangeshiftR package is not yet released and only available as beta version upon request. The official release is planned for May 2020. For further information please contact Anne or Damaris.
