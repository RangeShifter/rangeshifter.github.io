RangeShiftR

Team

Anne-Kathleen Malchow, Damaris Zurell

Project link

https://github.com/RangeShifter/RangeShiftR-package

Tutorial

https://rangeshifter.github.io/RangeshiftR-tutorials/

We here introduce the new RangeshiftR package, which makes RangeShifter 2.0 available from within R (Malchow et al. 2021). The R package makes it especially easy to run more complex simulations, it allows more coherent workflows and easier reproducability as well as integration with other R packages and plotting functions.

The RangeshiftR package is available on our github page. Analogous to the RangeShifter GUI, we provide tutorials to learn the different features of RangeshiftR using example applications from Bocedi et al. (2014, 2021) and Malchow et al. (2021). These cover some of the main features of RangeShifter, and help becoming familiar with the software.

For help on the RangeshiftR package, please visit our Forum pages to browse earlier topic or add new topic, or contact us directly through RangeshiftR@uni-potsdam.de.
